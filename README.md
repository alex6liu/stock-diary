# stock dairy

> An electron-vue sqlite3 project

#### Build Setup

``` bash
# 安装依赖
npm install

# 本地开发
npm run dev

# 本地打包（安装程序）
npm run build

# 本地打包（直接目录）
npm run build:dir

```

---

#### 项目

在[easy-invoices](https://github.com/CaanDoll/easy-invoices)的基础上进行修改

#### 目的

记录下每一笔交易， 并计算出年化收益， 看看能不能跑赢理财和货币基金。

#### 公式
年收益率=[（投资内收益 / 本金）/ 投资天数] * 365 ×100%

或

年收益率=[（（总值 - 本金） / 本金）/ 投资天数] * 365 ×100%
